## 🌿 브랜치 규칙

이름은 **영어 이름 또는 영문 표기만** 사용해주세요.  
예: `juhwan-kim`, `kim-juhwan`  

각자 자신의 브랜치에서만 작업하며,  
`main` 브랜치에는 **직접 push하지 않습니다.**

---

### 🔤 일반적인 브랜치 이름 규칙

- **소문자 사용:** 브랜치 이름은 모두 소문자로 작성합니다.  
- **하이픈(-) 사용:** 단어와 단어 사이는 하이픈으로 구분합니다.  
- **간결성:** 이름은 짧고 명확하게 작성합니다.  
- **의미 전달:** 브랜치의 기능이나 목적을 알 수 있도록 이름을 짓습니다.  

예시:  
`feature-login`, `fix-bug-image-upload`, `juhwan-kim`

---
